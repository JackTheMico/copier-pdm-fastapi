## Installation

With `pip`:

```bash
pip install copier-pdm-fastapi
```

With [`pipx`](https://github.com/pipxproject/pipx):

```bash
python3.8 -m pip install --user pipx
pipx install copier-pdm-fastapi
```

With `pdm`:

```bash
pdm install copier-pdm-fastapi
```
